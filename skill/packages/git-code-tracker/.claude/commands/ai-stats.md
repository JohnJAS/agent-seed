View AI code contribution statistics for this repository.

Run `node .claude/skills/ai-code-tracker/scripts/ai-code-stats.js $ARGUMENTS`

Present the results to the user in a clear summary. Highlight the AI contribution ratio and any notable trends.

Supported options:
- `--last N` — show stats for the last N commits
- `--author <name>` — filter by author
- `--since <date>` — filter by date (e.g. 2026-05-01)
